Calculator Program in C

A simple console-based calculator program written in the C programming language.
This project demonstrates the use of conditions, user input, and basic arithmetic operations.

Features:
1.Addition
2.Subtraction
3.Multiplication
4.Division

How to Compile and Run (using GCC):
Step 1: Compile the program:
gcc calculator.c -o calculator

Step 2: Run the program:
./calculator
